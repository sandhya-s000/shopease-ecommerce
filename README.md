# 🛒 ShopEase - E-Commerce Frontend

## 📖 Overview

ShopEase is a responsive E-Commerce Frontend developed using HTML, CSS, and JavaScript. It provides a modern shopping interface with authentication, product browsing, cart management, and checkout using Local Storage.

---

## 🚀 Features

- 🏠 Home Page
- 🔐 User Login
- 📝 User Registration
- 👤 User Profile
- 🛍️ Product Listing
- 🔍 Live Product Search
- 🛒 Shopping Cart
- 💳 Checkout Page
- 💾 Local Storage
- 📱 Responsive Design
- ✨ Modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
